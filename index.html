Il Demorianesimo

Il Demorianesimo è un culto che si basa sul software libero, la lista completa di software e linguaggi di programmazione accettati dal demor è visibile in questa pagina, alcuni software accettati dal Demor sono: OpenBSD, TempleOS.

Qualunque software proprietario è strettamente bandito dal Demorianesimo
